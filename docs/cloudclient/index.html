Skyclient installs things like mods and texture packs for a Minecraft profile (designed for the Hypixel server).

Cloudclient is a version of Skyclient, accessible online (hence the Cloud).

Your browser doesn't have any way to access directories. This means Cloudclient can't write to your profile. Try again with Chrome/Edge, or any Chromium-based browser.

      Installing

      What launcher would you like to install for?

      MultiMC Prep

      If you use MultiMC, first you need to create an instance for Skyclient.

      1. Click the "Add Instance" button
      2. Name the instance Skyclient and add it to a group if you want
      3. Set the Minecraft version to 1.8.9
      4. Set the icon to the Skyclient logo (click to download)
      5. Select the instance and click "Edit Instance"
      6. Click "Add Forge"

      Then, when you upload your .minecraft folder, choose the one you find from the "Open .minecraft" button.


      Now you need to let Cloudclient add mods, texture packs, and config to your .minecraft folder. You'll need to find it and put it here.

      Drop here
      Or click to browse

      By continuing, you agree to let Cloudclient modify your .minecraft folder, and acknowledge that it may overwrite existing mods, texture packs, or configuration. We are not liable if something goes wrong.

      Taking a backup is highly recommended. You can always close this tab or reload the page.

      Consider removing existing mods/texture packs/config in the directory you chose.
